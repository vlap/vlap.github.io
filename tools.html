Tools

A collection of tools that make my life easier... in the long run.

  1. LyX! With LaTeX and BibTeX, of course.
  2. Mendeley. For keeping my library organized across several machines and managing references. Export to BibTeX for use with Lyx. See Managing publication lists in HTML to find out how to semi-automatically convert the Bibtex file with your publications into HTML.
  3. Dropbox. For keeping all important documents synced and collaborative development. Altough I would like to make an effort and start using Github.
  4. writeLaTeX. For collaboratively writing papers. Heard good things about Bitbucket. They give you free private repositories.
  5. HTML and free CSS templates (from here or here). This website is based on this or this examples. However, Jekyll (uses Github) is a more convenient tool for building a notebook-like website.
